# User Stories Initiator

Der Initiator ...

```
...möchte den Kalender einsehen um offene Termine zu finden.
```

```
... möchte sich für einen Termin anmelden um seinen Vortrag zu halten.
```

```
... möchte in der Vortragsansicht sehen wie viele teilnehmen um das Publikum einschätzen zu können.
```

```
... möchte Nachrichten an Teilnehmer verschicken um so zusätzliche Informationen zu teilen.
```

```
... möchte Werbung auf einem schwarzen Brett machen um mehr Interesse zu erlangen.
```

```
... möchte auf dem schwarzen Brett Events erstellen können um die Gemeinschaft zu fördern.
```

```
... möchte das Event kategorisieren können, so dass Teilnehmer eine bessere Übersicht erhalten.
```

```
... möchte dem Event einen Treffpunkt/Standort hinzufügen, damit die Teilnehmer wissen wohin sie müssen.
```

```
... möchte Start und Endzeit festlegen um den Zeitrahmen festzulegen.
```

```
... möchte dem Event eine min. Teilnehmeranzahl vergeben, so dass es sich überhaupt lohnt das Event stattfinden zu lassen.
```

```
... möchte dem Event eine max. Teilnehmeranzahl vergeben, um das Event zu begrenzen.
```
