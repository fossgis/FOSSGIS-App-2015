# User Stories Redner/Workshopleiter

Der Redner/Workshopleiter ... 

```
...möchte Nachrichten an Teilnehmer seiner Veranstaltung verschicken um weitere Informationen zu teilen.
```

```
... möchte auf einer Karte sehen können wo sich der Veranstaltungsort befindet um einschätzen zu können wie weit seine Unterkunft entfernt liegt. 
```

```
... möchte einen Raumplan des Veranstaltungsorts einsehen können um sich dort zu orientieren. 
```

```
... möchte Kontaktdaten eines Ansprechpartners bzgl. seines Vortags erhalten um bei offenen Fragen Hilfe zu erhalten.  
```

```
... möchte einen Veranstaltungsplan einsehen können um andere Veranstaltungen besuchen zu können.
```

```
... möchte einen Veranstaltungsplan einsehen können um zu wissen wann der eigene Vortrag stattfindet. 
```

```
... möchte wissen mit welchen Nahverkehrsmitteln er zum Veranstaltungsort kommt. 
```

```
... möchte auf dem schwarzen Brett Werbung für seinen Vortrag machen um viele Zuhörer bei seinem Vortrag zu haben. 
```

```
... möchte eine Möglichkeit haben sich mit seinen Kontaktdaten zu präsentieren um interessierten Zuhörern eine Kontaktmöglichkeit zu geben.
```

```
... möchte ein Formular für Kommentare und Bewertungen über seinen Vortrag anbieten können um Feedback zu erhalten. 
```

```
... möchte Informationen über die Stadt erhalten um außerhalb der Veranstaltung etwas zu unternehmen. 
```
